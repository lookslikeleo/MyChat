# MyChat

## Project Overview

MyChat is a browser-based messaging app prototype built with React and TypeScript. The current version focuses on the front-end experience: first-time profile setup, a contact list, a focused one-chat-at-a-time conversation view, and a placeholder settings screen for future expansion.

Right now, the project is meant as a UI and flow foundation rather than a full real-time messenger. User profile data is stored locally in the browser, and the conversations are powered by sample data so the interface can be designed and tested before backend and messaging logic are added.
